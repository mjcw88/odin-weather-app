# The Odin Project (Full Stack Javascript) - Weather App
Weather App project for the Odin Project Full Stack Javascript course.

Full project specs can be found [here](https://www.theodinproject.com/lessons/node-path-javascript-weather-app). This project primarily focuses on asynchronous functions, API fetches with promises, and rendering/storing the data that is returned.
